## ZFS

Устанавливаем **zfs**

```console
[vagrant@otuslinux ~]$ cat /etc/redhat-release 
CentOS Linux release 7.8.2003 (Core)

[vagrant@otuslinux ~]$ sudo yum install https://zfsonlinux.org/epel/zfs-release.el7_8.noarch.rpm

[vagrant@otuslinux ~]$ gpg --quiet --with-fingerprint /etc/pki/rpm-gpg/RPM-GPG-KEY-zfsonlinux

```
